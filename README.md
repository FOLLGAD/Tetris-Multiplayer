# Tetris multiplayer
An online competetive version of Tetris. In action at: https://tentium.se/games

## Getting started
```
npm i
npm start
```

## Files
* server
  * game - Contains all game classes
  * index.js - Entry point
  * sockets.js - Socket.io sockets
